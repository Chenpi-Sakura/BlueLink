

# 蓝链 BlueLink 项目核心说明
## 一、功能定位与逻辑
1. **产品定位**
  端侧 AI 认知溯源引擎，面向科研、创作、深度学习，**AI 不生成答案，只做知识索引与原文溯源**。

2. **核心逻辑**
  拒绝 AI 直接投喂结论，引导用户回归原始文本，主动思考、自主验证，解决虚假获得感、信息冗余、隐私泄露问题。

3. **核心功能逻辑**
- **溯源检索**：用户提问 → AI 定位原文 → 输出锚点 → 跳转原文片段
- **信息降噪**：文档入库 → 识别重复内容 → 折叠冗余 → 只展示信息增量
- **费曼伴学**：用户复述知识点 → AI 检测认知偏差 → 引力线绑定错误与原文 → 自我校准
- **灵感图谱**：灵感/笔记入库 → 自动关联知识 → 构建个人可视化知识星图
- **隐私保护**：所有本地数据、灵感、知识库**端侧处理、本地加密，不上云**

---

## 二、界面设计理念
1. **设计核心**
  人文底色，极简透视；机器隐匿于文字之后，做提灯向导，不喧宾夺主。

2. **视觉理念**
- 回归纸质阅读质感，温润、低干扰、长时阅读友好
- 视觉降噪，突出信息增量与核心内容
- 古典阅读感 + 现代 AI 轻盈感平衡

3. **界面风格规范**
- **主色调**：米白色底 #FDFBF7（纸张质感）
- **强调色**：克莱因蓝 #002FA7（品牌与交互引导）
- **辅助色**：灰色 #C0C0C0（视觉弱化与降噪）
- **字体**：阅读区域用衬线体，交互区域用无衬线体
- **质感**：大圆角、多留白、微阴影、磨砂玻璃半透效果

4. **交互设计理念**
- **锚点穿梭**：不输出答案，只给路径，强制回归原文
- **聚光灯阅读**：目标高亮，其余弱化，不破坏阅读流
- **引力线关联**：错误/疑问直连原文，直观校准
- **模糊悬浮**：半透卡片，透而不漏，保持空间清爽

---

## 三、四大核心界面逻辑
1. **首页（文库聚合）**：瀑布流文献卡片，AI 提炼概要，悬浮快捷入口
2. **对话页（线索锚定）**：无气泡聊天，只展示问题、溯源锚点、引力线
3. **阅读页（动态透视）**：原文沉浸式阅读，增量高亮、冗余折叠、聚光灯效果
4. **图谱页（知识星图）**：知识点节点+关联线，可缩放聚焦，点击跳转原文

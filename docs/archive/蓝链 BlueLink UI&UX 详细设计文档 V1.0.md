## 1. 概述
本文档定义蓝链 BlueLink 客户端（uni-app）的界面与交互设计规范，涵盖信息架构、页面布局、组件行为及视觉表现。设计目标为在移动端高效呈现“溯源引导”与“降噪阅读”的双重体验，贯彻“人文底色，极简透视”的理念。

适用平台：Vivo(Android) App。

## 2. 设计原则与设计语言
### 2.1 设计原则
1. **路径优先，拒绝答案**：界面中永远不直接展示“最终答案”，一切信息输出均为引导性的线索（锚点、引力线、提问式引言）。
2. **降噪聚焦**：利用折叠、模糊、灰态等手法让用户注意力集中在当前核心任务上。
3. **一触即达**：关键操作（如跳转原文）不超过一次点击，缩短路径。
4. **主动可控**：用户始终能调节溯源颗粒度、提示明确度、折叠状态，保留掌控感。
5. **隐私可视**：私密数据在界面上有明确区分标识，让用户放心。

### 2.2 视觉语言
- **色彩**：米白基底 (#FAF7F2) 营造纸质阅读的温润感；克莱因蓝 (#002FA7) 用于所有可交互元素、高亮、引力线，形成品牌跳脱感；灰色 (#B0B0B0) 用于非焦点文本和弱化层。
- **字体**：内容区使用衬线体（中文：思源宋体/方正书宋；英文：Georgia），UI控件使用无衬线体（中文：PingFang SC，英文：SF Pro Text）。
- **形状**：大圆角卡片 (16px)、胶囊按钮 (24px)、磨砂玻璃 (backdrop-filter: blur(20px))。
- **阴影**：轻量弥散阴影 (0 2px 8px rgba(0,0,0,0.08))，表达层级但不干扰阅读。

## 3. 信息架构与导航
采用**底部标签栏**主导航，含四个核心视图：

1. **文库**（首页） — 瀑布卡片流，知识库概览与灵感快速入口。
2. **对话** — AI 伴学与概念提问，费曼模式切换。
3. **图谱** — 知识星图，全局关联视角。
4. **我的** — 设置、隐私、用户画像。

全局悬浮元素：  
- 文库页底部快捷捕获栏（+号展开语音/文字/拍照）。  
- 对话页底部常驻溯源粒度调节按钮。

## 4. 页面设计详述
### 4.1 启动与引导
- **首次启动**：展示三张引导卡片（可滑动），分别介绍“AI只提供锚点”、“隐私不外泄”、“阅读即思考”，配插图。最后一张提供“开始构建你的知识库”按钮。
- **隐私授权弹窗**：在引导结束后，弹出隐私选项：“仅本地模式”、“允许云端增强（脱敏）”、“了解更多”。用户必选其一。
- **空状态**：文库无内容时，中央显示一支克莱因蓝羽毛笔图标，文本“导入第一篇文献或记录一个灵感”。

### 4.2 文库首页（瀑布流卡片）
**布局**  
- 顶部：搜索框（可搜索本地全部文档与卡片），右侧“筛选”图标。
- 主区域：双列瀑布流卡片（根据内容高度自适应）。每张卡片包含：
  - 标题（衬线体，1.2em）
  - 自动提炼的概念路标（灰色小字，2行截断）
  - 来源与时间（底部灰色小字）
  - 隐私角标：若为私密，右上角显示小小的“锁”图标（克莱因蓝）。
- 卡片长按弹出操作菜单：打开、标为私密/公开、删除、查看关联图谱。
- 底部悬浮快捷栏（高度 64px，磨砂玻璃背景，距底部安全区 16px）：三个图标按钮（文本、语音、相机），中间主按钮为“+”号，点击展开三个选项，再次点击收起。

**交互**  
- 下拉刷新卡片排序。
- 点击卡片进入阅读器（增量折叠模式，若新文档尚未完成去重则显示“分析中”状态）。

### 4.3 对话溯源页（锚点穿梭）
**布局**  
- 顶部标题“对话”，右侧可切换“费曼模式”。
- 消息流：用户提问用浅灰背景圆角气泡（无衬线体），AI 回复为特殊卡片式，不使用传统气泡。
- **AI 回复卡片**设计：
  - 磨砂玻璃基底，圆角 20px，左侧边缘有一条克莱因蓝色竖线（引力线视觉延续）。
  - 顶部为引言文本（衬线体，16px，行高 1.8），最多三句话，若超过则折叠“...展开”。
  - 下方为锚点列表，每条锚点为一个横向卡片（白底，微弱阴影），包含：
    - 左侧：文档标题（加粗）+ 片段前20字（灰色）+ “跳转原文” 克莱因蓝文字按钮。
    - 右侧：相关性百分比圆环（可选）。
- 输入区：底部固定，左侧“粒度”选择器（文章级/句词级），中间输入框，右侧发送/语音切换按钮。

**交互**  
- 点击任一锚点卡片，页面整体向左滑动（Push 动画），进入阅读器并自动聚光灯高亮对应段落。
- 从阅读器返回时，该锚点卡片增加“已阅”勾选标记。
- 提问后若等待大模型返回，显示“正在溯源...”骨架屏，用脉冲动画表示。

**费曼模式切换**  
- 点击“费曼模式”按钮，对话顶部出现提示条“请用自己的话解释 XXX 概念”。
- 用户输入或语音后，AI 返回偏差报告卡片（同样磨砂基底），内包含偏差段落与对应引力线，每条偏差可点击直达原文。

### 4.4 智能阅读器（聚光灯 & 增量折叠）
**页面结构**  
- 顶部导航栏：返回按钮 + 文档标题（居中） + 书签/更多按钮。
- 正文区域（可滚动）：
  - 正常情况下全文按顺序展示，衬线体，1.8倍行距，段落间距 1.5em。
  - 折叠段落显示为一条 40px 高的提示条：“AI已折叠 327 字重复背景，点击展开”，左侧淡蓝色折叠线。
  - 增量段落左侧有淡蓝色边框标记（2px，透明度50%）。
- 底部悬浮“目录”按钮（可呼出滑动目录抽屉）。

**聚光灯模式**  
- 触发：由锚点跳转或费曼引力线进入时。
- 视觉效果：目标段落背景加一层半透明米白色块（或轻微发光），文字保持纯黑；其余段落全部添加 `opacity: 0.3` 并应用灰色文字（整体灰态但可读）。
- 目标段落左侧绘制一条垂直克莱因蓝线（引力线标识），并有一个小小的浮动标签显示跳转来源（如“来自对话提问”）。
- **退出**：点击屏幕任何灰态区域或点击顶部“退出聚光灯”按钮，灰态层淡出 (0.3s)，恢复正常阅读。退出后目标段落仍保留一个淡蓝色下划线表示已访问过。

**增量折叠模式**  
- 从文库进入时的默认模式：折叠条为默认折叠状态，阅读时用户可逐一点击展开。
- 提供“展开全部增量”浮动按钮（仅在折叠内容超过3处时显示），点击后所有折叠内容同时展开，按钮变为“折叠已知背景”。

### 4.5 知识图谱页
**视觉**  
- 背景色：深灰色 (#1E1E2E) 或纯黑，以突显节点与引力线的色彩。
- 节点采用三种形状/颜色区分：
  - 文献节点：淡蓝色圆形，带文档图标。
  - 灵感节点：暖黄色菱形，带灯泡图标。
  - 概念节点：克莱因蓝圆角方形，显示概念名。
- 连线（引力线）根据关系类型颜色不同：
  - 支撑关系：绿色虚线
  - 质疑关系：红色虚线
  - 补充关系：克莱因蓝实线
  - 引用关系：灰色点线
- 连线可有动态流动粒子表示方向（加载时播放）。

**交互**  
- 手势：双指缩放，单指拖动，双击重置视角。
- 点击节点：镜头平滑拉近并居中该节点，高亮其一阶邻居节点及连线，其余节点透明度降至 10%。
- 点击高亮区域外空白处，解除聚焦，恢复全图。
- 长按节点弹出菜单：查看原文、打开关联文档、从此节点开始提问。
- 底部栏包含“布局切换”按钮（力导向/环形）和“图例”按钮。

**数据加载**  
- 图谱页面初次加载时显示加载动画，随后节点按批出现（入场动画：从小到大、透明度渐现）。

### 4.6 灵感快捷捕获
**触发**：首页底部悬浮栏的 + 号，或在任何页面双指下滑手势（可配置）。
**捕获弹窗**（自底部滑出，高度约屏幕 1/3）：
- 顶部三个 Tab：文字、语音、拍摄。
- 文字 Tab：多行输入框，占位符“记录一个想法...”，下方“标签”输入框（自动提取关键词），“私密”开关（默认开启）。
- 语音 Tab：长按录音，松开发送，实时转文字显示（调用 Web API 或 uni 录音）。
- 拍照 Tab：调用相机，拍摄后可裁切，自动 OCR 提取文字。
- 最下方“收录为灵感卡片”按钮，点击后弹窗收起，卡片随即出现在文库瀑布流顶部（带一个小动画）。

### 4.7 设置与隐私
**页面布局**：  
- 用户头像区 + 昵称 + 专业描述（可编辑）
- **认知设置**：
  - 默认溯源粒度：文章级 / 句词级
  - 提示明确度滑块（温和启发 <----> 直指核心）
  - 探索深度开关（标准/深度）
  - 伴读语气选择（平实 / 鼓励 / 严肃）
- **隐私与安全**：
  - 当前模式显示：仅本地 / 本地+云端增强
  - 更改隐私模式（需再次确认）
  - 管理私密标记的文档列表
  - 数据导出与删除按钮
- **其他**：引导重播、关于、反馈。

## 5. 核心交互组件设计
### 5.1 锚点卡片
- 组件名：`AnchorCard`
- Props：`title`, `snippet`, `segmentId`, `score`, `isRead`
- 样式：白底，圆角12px，左侧2px蓝线，内部水平布局。右侧箭头图标。已读状态时卡片整体降低不透明度至0.6，箭头变为勾号。
- 点击事件：`emit('navigate', segmentId)`，同时添加涟漪动画。

### 5.2 聚光灯蒙版
- 实现方式：通过两个叠加的 `<view>` 层：
  - 底层为全文内容，使用 `v-for` 渲染，动态 class 绑定 `{ dimmed: !isSpotlight(segment) }`。
  - 顶层为高亮段落容器（绝对定位），通过 `scroll-top` 和 `getBoundingClientRect` 计算位置，显示原文并加上蓝色左边框和半透明白色背景。
- 退出时，蒙层透明度渐变为0，顶层消失。

### 5.3 折叠提示条
- 组件名：`FoldBar`
- Props：`foldedCount`, `expanded`
- 样式：高度 40px，浅蓝色虚线边框，内含文字“AI已折叠 327 字重复背景，点击展开”，右侧一个向下箭头。点击后箭头旋转180度，下方内容展开，提示文字变为“收起重复背景”。
- 展开动画：使用 `max-height` 过渡 (300ms)。

### 5.4 引力线动画
- 在阅读器聚光灯下，目标段落左侧绘制一条垂直克莱因蓝线，高度覆盖该段落，顶部有一个小圆点（5px），圆点带脉冲效果（缩放 opacity 从1到0.5循环）。
- 在费曼偏差报告中，偏差卡片的左侧同样有引力线，点击后跳转到原文段落时，该条线有一个粒子飞行动画（从卡片位置飞向目标段落）。

### 5.5 磨砂悬浮栏
- 组件：底部快捷栏、半屏锚点列表容器、设置页面部分区块。
- 使用 `backdrop-filter: blur(20px)` + 半透明白色背景 (`rgba(255,255,255,0.7)`) 实现。
- 注意小程序兼容性：微信小程序支持 `backdrop-filter` 需基础库 2.10.0+，条件编译时 App 端全支持。

## 6. 动效与反馈
- **页面转场**：从对话页跳转至阅读器采用 `slide-left` 动画（300ms），返回时 `slide-right`。
- **锚点卡片加载**：骨架屏使用闪烁脉冲，模仿卡片形状。
- **按钮点击**：缩放至 95% 然后回弹 (spring 动画)。
- **图谱节点出场**：节点从中心依次飞入，带弹性缓动。
- **聚光灯模式切换**：灰态和蓝色边缘以 0.25s fade 呈现，自然不刺眼。
- **快捷栏展开**：+ 号按钮旋转45度，三个子按钮从下方向上弹出依次出现（stagger 50ms）。

## 7. 适配与无障碍
- **字体缩放**：响应系统字体大小设置，使用 `rem` / `rpx` 单位。
- **暗黑模式**：初版暂不考虑，但色彩系统预留暗黑适配变量（米白改为深灰，文字反色）。
- **读屏支持**：为所有非文本元素添加 `aria-label`（uni-app 中通过 `aria-label` 属性），如“跳转到原文段落”、“私密文档锁定图标”。
- **安全区域**：底部栏、快捷栏使用 `safe-area-inset-bottom` 适配 iPhone X+。
- **横屏**：阅读器支持横屏（强制横屏或横屏时切换为双栏布局，左侧原文、右侧锚点/笔记，比赛阶段可选做）。
